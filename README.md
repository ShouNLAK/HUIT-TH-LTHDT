## 🚀 Hướng dẫn sử dụng

### 1. Clone Repository
```bash
git clone https://github.com/ShouNLAK/HUIT-TH-LTHDT.git
cd HUIT-TH-LTHDT
```

### 2. Mở dự án
- **Visual Studio**: Nhấp đúp vào file `.sln` trong thư mục bài học
- **Visual Studio Code**: Mở thư mục chứa project

### 3. Build & Run
```bash
# Trong Visual Studio
Ctrl + Shift + B  # Build
F5               # Run

# Hoặc dùng terminal
dotnet build
dotnet run
```
